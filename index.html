Forms

  • Acceptable Parameters for 'type' in Input tags
    [ 'button', 'checkbox', 'color', 'date', 'datetime', 'datetime-local', 'email', 'file', 'hidden', 'image', 'month', 'number']

  • 'Method' parameter contains the type of requestmade to the actions' endpoint

  • 'Action' parameter contains the 'endpoint' to which the POST request has to be made.

Code:

<!DOCTYPE html> <html lang="en"> <head> <title>Forms</title> <style> .formbody{ padding-top: 1cm; align-items: center; font-family: 'Comic Neue', cursive; background-color: whitesmoke; background-image: url('https://rb.gy/wcrqyy'); } .inputform{ width: 10.5cm; height: 1cm; } .formMain{ border: 1px solid black; padding: 1cm; background-color:brown; opacity: 0.7; color: white; font-size: 20px; } input:hover{ border: 1px #1e3f5a solid; } </style> </head> <body> <div class="formbody"> <form class="formMain" action="#" method="POST"> <!-- Text Input --> <label for="name">Name</label><br> <input type="text" name="name" id="text" placeholder="name"><br><br> <!-- Radio Button --> <label>Gender</label><br> <input type="radio" id="gender" name="gender" value="male"> <label for="male">Male</label><br> <input type="radio" id="gender" name="gender" value="female"> <label for="female">Female</label><br> <input type="radio" id="gender" name="gender" value="other"> <label for="other">Other</label><br><br> <!-- Dropdown --> <label for="branch">Choose Your Branch:</label> <select id="branch" name="branch"> <option value="CSE">CSE</option> <option value="IT">IT</option> <option value="CE">CE</option> <option value="ETC">ETC</option> <option value="EEE">EEE</option> </select><br><br> <!-- Checkbox --> <label >Select your Choice of Clubs in Tech Soc</label><br> <input type="checkbox" id="checkbox" name="psoc" value="tech"> <label for="Tech"> Programming Society</label><br> <input type="checkbox" id="checkbox" name="tars" value="cultural"> <label for="cultural"> Robotics Society</label><br> <input style="align-self: center; width: 2cm; color: brown; background-color: rgb(255, 255, 255);" type="submit" placeholder="SUBMIT"> </form> </div> </body> </html>
